# commentFrameGenerator
Generate a frame for the given string in input

## Usage

```
$> c++ foo.cpp -o genrator
```
### Input 
``` bash
$> ./genrator Hey Mate Methods Variables "Operator Overloads"
/* -------------------------------------------------------------------------- */
/*                                    Hey                                     */
/* -------------------------------------------------------------------------- */
/* -------------------------------------------------------------------------- */
/*                                    Mate                                    */
/* -------------------------------------------------------------------------- */
/* -------------------------------------------------------------------------- */
/*                                  Methods                                   */
/* -------------------------------------------------------------------------- */
/* -------------------------------------------------------------------------- */
/*                                 Variables                                  */
/* -------------------------------------------------------------------------- */
/* -------------------------------------------------------------------------- */
/*                             Operator Overloads                             */
/* -------------------------------------------------------------------------- */
```
